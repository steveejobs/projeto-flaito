# STACK - Projeto Flaito

Este documento lista as tecnologias e padrões técnicos utilizados no projeto.

## Core Runtime & Frameworks
- **Runtime**: Bun / Node.js
- **Frontend Framework**: React 18+ com Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS / Shadcn/ui

## Backend & Infrastructure
- **BaaS**: Supabase (Database, Auth, Storage, Edge Functions)
- **Database**: PostgreSQL (via Supabase)
- **Deployment**: Coolify (na VPS atual)

## Integrations
- **WhatsApp**: Z-API
- **Business Logic**: Módulos dedicados para Médicos e Advogados

## Development Tools
- **AI Engine**: GSD (Get Shit Done) Framework
- **Task Management**: ROADMAP.md dinâmico

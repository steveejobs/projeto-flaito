# Projeto Flaito

## Sobre

Sistema completo de gestão para escritórios de advocacia.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Como rodar localmente

```sh
npm install
npm run dev
```

## Deploy

O deploy é realizado automaticamente via Coolify usando o Dockerfile incluído no repositório.
